HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML elements work together to buidl the frame of a webpage
  • As a developer you should think of HTML First!
  • HTML elements are often composed of opening and closing tags represented by angled brackets

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to stle a webpage using css; inline css, internal css style sheet, and external css style sheet.
  • The most common way to style a webpage is by using an external css style sheet.
  • External style sheets are so popular among developers because it allows them to keep their css elements seperate.
  • Developers use inline css if they want to style an element directly in the HTML File.
  • Inline is only used for quick and permanent changes since they are less flexible.
  • Internal CSS style sheets contain css rules in the head section of the HTML File.
  • Internal CSS style sheets rules only apply to that specific webpage where you can configure css classes and ids.
